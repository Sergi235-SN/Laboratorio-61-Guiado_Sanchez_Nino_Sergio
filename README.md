# Control por Voz con AssemblyAI y Python

Este proyecto permite grabar audio desde el micrófono, subirlo a la API de AssemblyAI y transcribirlo a texto en español. Además, incluye una interfaz gráfica en Python para ejecutar comandos de voz que interactúan con un “motor” simulado en la GUI.

## Comandos de Voz

| Comando de voz       | Acción en la aplicación                    |
|---------------------|-------------------------------------------|
| `enciende`           | Enciende el motor                          |
| `apaga`              | Apaga el motor                             |
| `salir`              | Cierra la aplicación                        |
| (sin voz detectada)  | No se detectó audio                         |
| Otro texto           | Comando no reconocido                        |
